# 我是懂球王 MVP

这是基于《我是懂球王》PRD 搭建的首版仓库，用于验证最关键的产品闭环：

- 首页：今日比赛列表、站队入口、预言卡、我的状态、榜单预览
- 比赛详情页：站队题 + 预测题提交、提交记录、赛后自动结算展示
- 我的懂球卡：懂球分、命中率、连胜、标签、荣誉称号
- 排行榜页：今日榜 / 本周榜 / 连胜榜 / 命中率榜
- 最小后台页：比赛状态查看、结算重试

当前仓库先落地一个可直接打开运行的静态 MVP 原型，并附带 Supabase 推荐表结构，方便后续继续接真实数据源与登录系统。

## 当前文件

- `index.html`：单文件 MVP 原型
- `supabase/schema.sql`：推荐数据库结构
